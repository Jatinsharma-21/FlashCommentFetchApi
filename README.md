# Comment Search API

This Flask API allows you to search and filter comments from a remote API based on various parameters such as author, date, likes, replies, and text.

## Getting Started

### Prerequisites

- Python 3.x
- Flask
- Requests library

### Installation

pip install -r requirements.txt

### Run the Flask application
python app.py
